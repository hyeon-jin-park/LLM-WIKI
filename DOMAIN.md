# Domain: User-Defined Knowledge

LLM WIKI does not ship with a fixed knowledge domain. The user defines the domain by importing source material and choosing one of six generic page types.

The product manages source files, reviewed Markdown pages, links, verification metadata, and maintenance history. It does not provide domain facts, cloud synchronization, automatic truth verification, or autonomous publishing.

This boundary makes the same harness reusable for study notes, research, recipes, product documentation, personal finance references, travel planning, or another user-owned knowledge domain.
